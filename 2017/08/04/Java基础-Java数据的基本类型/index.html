Java基础-Java数据的基本类型

发表于 2017-08-04 | 分类于 Java基础 |

摘要：变量在内存中进行储存时，首先需要申请空间，内存管理器是根据变量类型来进行分配空间的。分配的空间只能用来存储该类型的数据。
因此通过定义不同的数据类型，可以在内存中存放小数，整数，字符。

了解

了解数据的基本类型前，我们先来了解一下位、字节、字符、字符集、编码的意思；以及字节、位、bit、byte、KB、B、字符之间的关系；

  • 位（bit）：是数据存储中的最小单位，每个二进制中的0或者1就是一个位。

  • 字节（byte）：8个位组成一个字节。

  • 字符：a、A、中、+………都表示一个字符。

  • 字符集：各个字符的集合。按照不同的标准收纳的字符集合。

  • 编码：规定每个“字符”分别用一个字节还是多个字节存储，用哪些字节来存储，这个规定就叫做“编码“。

    1字节 = 8位 即为 1byte = 1bit 也是 1B = 8b

    1字符 = 2字节 = 16位

    1KB = 1024B

Java中有俩大数据类型：分为:

  • 基本数据类型

  • 引用数据类型

基本数据类型

  • byte
    占8位，带符号的整数。

    最小值：-2^7 = -128

    最大值：2^7-1 = 127

    默认值：0

  • short
    占16位，带符号的整数。

    最小值：-2^16

    最大值：2^16-1

    默认值：0

  • int
    占32位，带符号的整数。

    最小值：-2^32

    最大值：2^32-1

    默认值：0

  • long
    占16位，带符号的整数。

    最小值：-2^64

    最大值：2^64-1

    默认值：0

  • float
    占32位，单精度浮点数。不能用来表示货币。

    默认值：0.0f

  • double
    占64位，双精度浮点数，是浮点数的默认类型，也不能用来表示精确的值，如货币。

    默认值：0.0d

  • boolean
    占1位，只有两个值true/false

    默认值：false

  • char
    char是一个单一的16位的Unicode字符；

    最小值：\u0000

    最大值：\uffff

引用数据类型

引用数据类型从字面意思上来看，其实就是引用类型指向一个对象，指向对象的变量是引用变量。这些变量在声明的时候被指定为一个类型。变量一旦声明后，类型就不能被改变。
如下：都是引用数据类型。且所有引用类型的默认值都是null。

  • 类
  • 接口
  • 数组

Java常量

Java常量一般用final 关键字来修饰，在程序运行时是不能被改变的。
声明如：final double PI = 3.1415926;
常量名一般为大写。是为了更好的区分。
