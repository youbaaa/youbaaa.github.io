Java之IO流

发表于 2018-06-15 | 分类于 Java中级

摘要：数据的传输即为I/O操作。

按照I/O流动方向，我们分为输入流和输出流。流入内存为输入（Input），流出内存为输出（Output）。

按照数据类型，我们分为字节流和字符流。字节流是以字节为单位，字符流是以字符为单位。

顶级父类如下：

- 输入流 输出流
字节流 InputStream OutputStream
字符流 Reader Writer
